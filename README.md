# alx_backend_graphql_crm# ALX Backend GraphQL CRM

This is a simple CRM (Customer Relationship Management) project built with Django and GraphQL using `graphene-django`. It demonstrates how to create a GraphQL API with Django models and queries.

---

## 📦 Tech Stack

- Python 3.12+
- Django 5.2+
- Graphene-Django
- Django-Filter
- GraphQL
- SQLite (default DB)
